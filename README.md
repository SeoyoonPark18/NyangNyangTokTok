+ 치매를 재미있게 예방할 수 있도록 게임을 통해 인지기능을 향상시키도록 함
+ 치매 예방에 적합한 연령층인 '60대'의 관심사를 반영한 컨셉의 게임으로, 카페의 직원이 되어 고양이 손님이 주문한 음식을 맞게 만들면 성공!

+ 게임 구성
  - 메뉴 외우기 > 메뉴 고르기 > 가격 계산하기 > 메뉴 만들기 > 전달하기 순서로 구성 됨
 
+ 총 5단계의 난이도로 구성
  - 수습기간 알바생 > 알바생 > 우수 알바생 > 부 매니저 > 매니저
